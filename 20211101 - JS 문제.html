1. 문제 
        사용자에게 숫자나 문자 아무거나 입력받아서 입력받은
        값이 숫자인지 문자인지 구분하는 조건문을 작성

        2. 문제
        사용자에게 다섯번의 값을 입력 받고
        다섯번의 입력이 모두 숫자인 경우에만 합을 출력
        입력값중 숫자가 아닌지 찾아내기

        3. 문제
        반복문을 사용자 입력을 두 번 받고 사칙연산을 사용
        더하기 빼기 곱하기 나누기 기능을 각각 함수로 정의
        eval함수는 사용하지 않는다.
        입력된 값이 정수인지 아닌지 확인 필요
        ex) 2입력 4입력 곱하기 -> 결과는 8 출력
