자판기 커피의 종류는 블랙커피 믹스커피 아이스커피 이고
        각각 1000원 500원 1500원 이다
        사용자에게 메뉴 이름을 입력받아 커피를 주는 자판기 코드 작성

        메뉴 선택 전 자판기에 사용자에게 돈을 입력 받는다
        주문한 커피의 가격이 넣은 돈 보다 작을 경우 결제완료 출력
        아니면 결제 실패 출력

        블랙커피는 수량 3개 -> 돈이 충분하면 결제 완료 출력
        믹스커피는 수량 0개 -> 재고 없음 출력
        아이스커피는 수량 5개 -> 돈이 충분하면 결제 완료 출력
