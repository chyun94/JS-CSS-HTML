Default Doucment 
웹페이지는 html,css,js 구성된다.
html: 구조를 만드는 마크업
css: 구조를 꾸미는 역할
js: 동적인 기능을 구현하는 프로그래밍 언어

동적인 기능: 사용자의 행동에 반응하는 기능들
마우스를 통한, 키보드를 통한

1. 사용자의 입력 및 계산
HTML 폼은 입력창만 제공한다.
입력을 받고 계산하는 기능은 수행할 수 없다.
키보드나 마우스 입력, 계산 등의 이벤트는 JS로 가능

2. 웹 페이지 내용 및 모양의 동적 제어
js 코드로 html 태그의 속성이나 컨텐트,css
프로퍼티의 값을 변경하여 웹 페이지의 동적 변화를 만든다.
* 프로퍼티는 css 속성(패딩 마진 보더 등등)

JS 코드의 입력 위치는? 

1. html문서 안의 body의 끝나는 부분에다 입력을 해도 무관
    why? script를 읽는 속도가 느리기 때문에 부담을 줄이기 위해서
    위에다가 입력시 script가 읽히기 전까지 페이지가 뜨지 않는다.

2.  HTML 태그의 이벤트 리스너 속성에 작성(onclick같은것)
    HTML 태그 내부에 클릭이 발생하면 동작할 소스코드를
    HTML 태그의 속성 중 하나인 이벤트 리스너 속성에 직접 작성.





이벤트 리스너

이벤트 리스터: 특정 태그에 이벤트가 발생하면 알아챌 수 있도록
속성을 지정해 놓는 것
대표적인 이벤트 리스너: onclick, omnouseover, onmouseout

3. html코드




자바스크립트 다이얼로그

다이얼로그: 사용자의 입력 및 메세지 출력
1. 프롬프트 다이얼로그
    프롬트 함수 사용:     prompt("재료");
    재료: 출력하려는 메세지
    기능: 사용자로부터 문자열을 입력받아 리턴한다
*return = 반환 되어서 나오는 값을 말함. 즉 재료를 입력시 나오는 결과물을 뜻함.

2. 확인 다이얼로그
    confirm()함수를 사용한다.
	()안에 재료, 출력하고자 하는 메세지
	기능: 사용자가 확인 / 취소를 누를 수 있는 기능
	확인을 누르면 true 를 리턴한다
	취소를 누르면 false 를 리턴한다

3. 경고 다이얼로그
    alert()함수를 사용한다
	()안에 재료로 출력하고자 하는 메세지 입력
	기능: 재료로 입력한 값을 단순히 출력해 전달


자바스크립트의 데이터 타입과 변수

식별자란 자바스크립트 개발자가 자바스크립트 프로그램의
변수, 함수 등에 붙이는 이름이다.

#시작자 선언 규칙.
1. 첫번째 문자 - 알파벳(대,소문자), 언더스코어
2. 두번쨰 이상 문자 - 알파벳, 언더스코어, 0~9까지의 숫자
3. 식별자는 대문자와 소문자를 구분한다.


자바스크립트 데이터 타입

자바스크립트 프로그래밍 언어로 다룰 수 있는 데이터의 종류.
1. 숫자: 1,10,100 etc..
2. 논리: true,false(참 거짓)
3. 문자열: ""로 감싸져 있는 모든 문자





 자바스크립트의 변수

변수는 자바스크립트 코드가 데이터를 저장하는 공간의 이름
