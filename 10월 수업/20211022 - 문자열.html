write 함수는 문자열 출력이 가능하다.
write 함수는 숫자 출력이 가능하다.
write 함수는 html 태그 출력이 가능하다 (문자열로 묶어준다"")

식과 연산자

식을 계산해서 결과를 얻는 과정: 연산
연산자의 종류
1. 산술연산: + - * / %
2. 증감연산: ++ --
3. 대입연산: =
4. 비교연산: < > <= >= == !=
5. 논리연산: && || !
