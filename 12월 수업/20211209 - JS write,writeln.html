document.write() 사용시 주의점
        HTML 텍스트를 HTML 문서에 추가하기 위해 document.write()를 사용한다.
        하지만 HTML 문서가 로드ㅇ되어 출력이 모두 이루어지고 나면 document객체가
        닫히기 때문에, 더이상 HTML 텍스트를 추가 출력 할 수 없다.

        document객체가 닫힌 후 document.write()가 실행되면 브라우저는
        document 객체에 담긴 현재 문서를 지우고 빈 document를 새로 연다.
        그래서 HTML 문서가 완전히 로드된 후 document.write()가 실행된다면, 현재
        HTML문서가 지워지고 새 문서가 작성되는 결과가 된다.

 DOM 객체의 동적 생성

        document.createElement("태그이름"); 
        이것을 호출하면 HTML 태그의 DOM 객체를 생성할 수 있다.

        
DOM 객체의 삽입

        appendChild() 메소드를 이용하면 객체를 DOM 트리에 삽입 가능하다.

        
DOM 객체의 삭제

        removeChild() 메소드를 이용하면 부모에게서 자식 객체를 떼어낼 수 있다.
        var removeObj = 부모.removeChild(떼어내고자 하는 자식 객체);
