총 10종의 강아지 사진을 보고 품종을 맞추는 퀴즈

        1. 웹 페이지가 로드되면 10종의 사진 중 무작위로 
            한 개 선정 후 화면에 출력
        2. 정답을 입력하는 input 태그 존재
        3. 다음 버튼 누르면 다음 무작위로 선정한 강아지 
            사진으로 이미지 변경
        4. 정답은 입력 인풋 태그에 입력하고 정답 버튼을 
            누르면 사진에 출력된 품종이 입력한 텍스트와 
            일치하는지 체크
        5. 정답을 인풋태그에 입력하지 않고 다음을 누르면 
            문제를 패스하는 것으로 처리

랜덤 이미지 출력
