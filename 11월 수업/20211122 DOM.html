DOM트리 특징
1. DOCUMENT 객체는 DOM 객체의 최상위에 있다.
2. HTML 태그에 의해 생성된 HTML 객체는 HEAD와 BODY 두 자식을 가진다.
3. DOM 트리의 루트는 DOCUMENT객체이다. 하지만 DOCUMENT객체는 DOM객체가 아니다.
4. DOM 객체의 종류는 HTML태그의 종류만큼 있다.
5. HTML 태그 하나 당 DOM 객체가 하나씩 생성된다.
(같은 태그가 여러개 있다고 해도 각 태그마다 DOM객체가 생성)
6. HTML 태그의 포함 관계에 따라 DOM 객체들이 부모 자식의 계층 관계로 구성된다.

DOM 객체들 사이의 관계
DOM 객체들은 DOM트리에서 부모 자식 형제의 관계로 연결된다.
parentElement 프로퍼티 = 부모객체
children 프로퍼티 = 직계 자식들의 컬렉션(배열)
firstElementChild = 첫번째 자식
lastElementChild = 마지막 자식
previousElementSibling 프로퍼티 = 왼쪽 형제 객체
nextElementSibling 프로퍼티 = 오른쪽 형제 객체
span태그3
DOM 객체의 주요 공통 프로퍼티, 컬렉션, 메소드
1. id 태그의 id속성 값
2. style 객체정보
3. title 태그의 title속성 값
4. tagName 태그이름 문자여르 항상 대문자로 표시됨(DIV,SPAN)
5. childElementCount 자식 DOM 객체 개수
6. firstElementChild 첫 번째 자식 객체
7. lastElementChild 마지막 자식 객체
8. nextElementSibling 다음 형제 객체(오른쪽)
9. previousElementSibling 이전 형제 객체(왼쪽)
10. parentElement 부모객체
11. offsetHeight 패딩 테두리 등을 포함한 높이
12. offsetWidth 패딩 테두리 등을 포함한 폭
13. offsetLeft 객체의 출력위치, 수직 옵셋
14. offsetTop 객체의 출력위치 수직 옵셋
