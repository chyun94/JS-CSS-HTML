Date 객체

        Date객체는 시간 정보를 담는 객체
        new키워드로 생성
        1. var now = newDate()  - 객체화

        2. var start = new Date(2021,10,5);
        
        
Date 객체의 주요 메소드

        getFullyear(): 2021과 같지 네자리 연도 리턴
        getMonth(): 0~11사이의 정수를 리턴
        getDate(): 한 달 내의 날짜 리턴(1~31)
        getDay():한 주 내 요일을 정수로 리턴(일요일=0,토요일=6)
        getHours(): 0~23사이의 정수 시간 리턴
        getMinutes(): 0~59사이 정수 분 리턴
        getSeconds(): -~59사이 정수 초 리턴
        getTime(): 1970년 1월1일0시0분0초 기준으로 현재까지 경과된 밀리초 단위 리턴
        getMilliseconds(): 0~999사이의 정수 밀리초 리턴
