식과 연산자

식을 계산해서 결과를 얻는 과정: 연산
연산자의 종류
1. 산술연산: + - * / %
2. 증감연산: ++ -- 변수의 앞 또는 뒤에 사용할 수 있으며 1씩 증감 가능.
		연산자가 변수의 앞: 전위 연산자
		연산자가 변수의 뒤: 후위 연산자
		++가 변수 앞에 있거나 뒤에 있거나 1증가는 공통사항.
		전위 연산자는 선 연산 후 대입 a=++b -> a에 먼저 들어가기 전에 b가 먼저 ++ 되고 a에 들어간다.
		후위 연산자는 선 대입 후 연산 a=b++ -> a에 먼저 들어가기전에 +된 상태에서 a에 들어간다.

3. 대입연산: = += -=
	= : 우측에 있는 식을 좌측에 대입
	대입연사자는 속도가 빨라서 사용한다.

4. 비교연산: < > <= >= == !=
	연산의 결과값이 true 아니면 false가 나온다
	

5. 논리연산: && || !
