문자열의 길이

    문자열의 길이는 배열의 길이처럼 length 프로퍼티가 존재한다. var hello = new String("HELLO");
    var x = hello.length;
    
문자열의 인덱싱

    문자열은 배열처럼[] 연산자를 통해 인덱싱이 가능하다.
    
String 객체 메소드

    1. concat(재료문자열) : 현재 문자열 뒤에 재료문자열을 연결한 결과를 리턴.
    2. indexOf(n) : 문자열 n이 처음으로 나타나는 인덱스의 값을 리턴한다.
    3. split("") : 빈칸을 기준으로 문자열을 분할하고 그 결과를 배열로 리턴한다.
    4. replace("a","b") : 문자열에서 a를 찾아 b로 변경한다.
